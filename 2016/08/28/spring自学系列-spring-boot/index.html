spring自学系列 - spring boot

简介

spring boot 大量简化了spring程序的简化配置，提供开箱即用的骨架程序

开发环境

IDE : IntelliJ IDEA 2016.2.1(64)
JAVA : 1.8.0_101

功能

元属性

@SpringBootApplication // same as @Configuration @EnableAutoConfiguration @ComponentScan

静态内容

spring boot 默认从以下几个地方读取静态文件

  1. /static
  2. /public
  3. /resources
  4. /META-INF/resources
    使用了spring mvc的ResourceHttpRequestHandler所以也可以在WebMvcConfigurerAdapter重写addResourceHandlers方法来自定义
    spring mvc会默认寻找index.html作为起始页
