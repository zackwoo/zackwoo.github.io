spring自学系列 - spring boot

简介

spring boot 大量简化了spring程序的简化配置，提供开箱即用的骨架程序

开发环境

IDE : IntelliJ IDEA 2016.2.1(64)
JAVA : 1.8.0_101

功能

元属性

@SpringBootApplication // same as @Configuration @EnableAutoConfiguration @ComponentScan
在引用程序main方法标注@SpringBootApplication 等同于同时标注了 @Configuration @EnableAutoConfiguration @ComponentScan

log 日志

日志文本输出
配置 application.properties 添加 logging.file or logging.path
日志文件每10M会自动分割
application.properties日志级别设定属性[logging.level.包名]

在代码中使用log

程序开始执行时运行初始化代码

可以继承 ApplicationRunner or CommandLineRunner
实现run方法

【烘焙】黄油曲奇

材料

  1. 黄油 70克
  2. 低筋面粉 84克
  3. 白砂糖 34克
  4. 淀粉 17克
  5. 盐 1克
  6. 鸡蛋 1个

操作

  1. 黄油室温软化，可以用刮刀均匀抹平即可
  2. 将白砂糖，淀粉，盐混合均匀分3次倒入软化黄油，每次都要将粉末搅拌入黄油看不到粉末为止
  3. 鸡蛋打匀
  4. 将蛋液分3次倒入黄油，倒入后都要搅拌均匀
  5. 黄油打发到发白微微膨化
  6. 将面粉筛入黄油用刮刀搅拌均匀，不要打发过度，看不见粉末为宜
  7. 准备好保鲜膜将面糊倒入，捏成正方长条后放入冰箱速冻20分钟
  8. 烤箱提前预热，上200度，下180度
  9. 将面团拿出切成5mm左右厚度的片，放进烤盘
  10. 烤大约15分钟完成

冰箱速冻时间可以久一点太早拿出来太软很难切片
面粉少20克换成可可粉可以做巧克力味

windows 环境下搭建jekyll

jekyll 将纯文本转化为静态网站和博客。
jekyll + github可以满足我们一般的日常对blog的基本要求了，下面是在windows环境下如何搭建jekyll的步骤。

OS : window7 x64

  1. 安装ruby（2.3.1）你可以根据需要安装最新版本 下载
  2. 安装ruby DEVELOPMENT KIT 根据安装的ruby版本下载相应的版本 下载
  3. 安装jekyll(3.1.6) 打开CMD命令行运行命令 gem install jekyll
  4. 在指定的文件夹中运行 jekyll new . 命令来创建一个jekyll网站
  5. 在github里创建一个“<用户名>.github.io”项目 将刚才jekyll创建的网站上传到该项目下
  6. 写新文章只需在_post文件夹下新建一个markdown文件
  7. 直接访问 https://<用户名>.github.io/ 完工

是不是非常的简单呢？
jekyll的自由度相当高，具体可 参考jekyll中文网站
